## The applications goal is to

- Quick search a stock to show quote
- Show todays top 10 trending stocks
- Show a stocks news

## Getting Started

1. cd frontend
2. npm install
3. npm run start

## API List

- IEX Cloud / https://iexcloud.io
- News Api / https://newsapi.org
